# StyleForge Lite v0.2

Phone-first update.

## New in v0.2

- Mobile layout improved.
- Track cards scroll horizontally on small screens.
- Drum grid is now 4 bars long.
- Bars alternate background shade.
- Sticky drum labels.
- Start/end grid scroll buttons.
- MIDI export now exports 4 bars.

## Run

```bash
cd styleforge-lite-v0.2
python -m http.server 8080
```

Open:

```text
http://127.0.0.1:8080
```
